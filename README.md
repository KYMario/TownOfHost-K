## この Mod について

この Mod は非公式のものであり、この Mod の開発に関して Among Us の開発元である"Innersloth"は一切関与していません。<br>
この Mod の問題などに関して公式に問い合わせないでください。<br>

この Mod はTOHに役職や機能など追加したModです
TOH-Kで起きたバグは本家や他MODには報告しないでまずはkに報告してください。<br>

また、TOH-Kに関する質問、バグ報告は基本、[DiscordServer](https://discord.gg/5DPqH8seFq)でのみ受け付けております。<br>

機能面などを開発者にDMで聞くのは超大迷惑なので基本おやめください。<br>

## 公開ルームについて
> [!Note]
> 現在のバージョンではAmongUs公式よりバニラサーバーで公開ルームを開くことができません。

> [**Note**]
> - 【理由】
> - 以前からHostMod(Moed含む)部屋での仕様, 不具合がAmongUs公式に問い合わせが多数寄せられた事が主な原因と捉えています。
> - また、ユーザーが**意図しないMod体験**が行われないよう注意喚起をしています。
<br>
-これは永続的な変更では有りません。<br>
-非公開ルームへの影響は有りません。<br>
-カスタムサーバーの影響は有りません。<br>
●公開ルームが開けない等のお問い合わせはご遠慮ください

## 勧誘部屋を開く又は勧誘する,勧誘からの参加は厳禁です。
**禁止行為**
> [**Note**]
> - ・非公開でMod部屋を建て,その部屋へのコードを公開ルームで伝え公開ルーム利用者がその部屋に参加する可能性がある行為。
> - (例)
> - 部屋主がサブ垢等で公開ルームを建てそこから非公開ルームへ勧誘する行為
> - 参加者がサブ垢等で公開ルームを建てそこから非公開ルームへ勧誘する行為
> -　部屋主,参加者が公開ルームへ参加し、そこから非公開ルームへ勧誘する行為

=>　公開ルーム利用者が意図しないMod体験を行われないようにするためです。<br>
**禁止されない行為**
> [**Note**]
> - Discordや村のオープンチャットで非公開ルームのコードを伝え身内間でMod部屋を開催する行為
> - Town Of Host Kのマッチメイキングで募集する行為
> - Youtubeなどの配信サイトで上記の禁止行為に該当しない参加型/身内村配信

=>　要するに**Modだと認知していない野良のプレイヤーが交わる可能性がある**とアウトです。<br>
<br>
勧誘部屋に遭遇した場合は勧誘先に無闇に参加せずまずは開発者にお知らせください。厳正に対処いたします。<br>

## TOH-Kでの処罰、利用不可について
v.5.1.61.1以降より[SuperNewRoles](https://github.com/SuperNewRoles/SuperNewRoles/)様等一部のModに搭載されていた<br>
ブラックリストをTown Of Host-Kでも適応するようにいたしました。<br>

> [!note]
> - 【処罰に値する利用方法】
> - ・上記に記載している勧誘部屋,勧誘行為に値する行動を行った場合
> - ・開発者が不適切かつ処罰するに値する迷惑行為を行った場合

上記に当てはまるプレイヤーはMod開発者の判断によりブラックリストに追加されます。<br>
**ブラックリストを適応しているMod部屋に永久もしくは一時的に参加/主催することが出来ません。**<br>

**★また、ブラックリスト追加により,ルーム作成にフレンドコードが必須となりました。ご了承ください。★**<br>

-----------------------------------------------

**【絶対に読むんだ】**<br>
Town Of Host-Kの利用をもって"処罰に値する利用方法"内で記載されている<br>
行動をしないと誓い,<br>
処罰に値する利用を行った場合はいかなる場合でも処罰を受け入れるものとします。<br>

-----------------------------------------------

要するに<br>
上の【処罰に値する利用方法】の行動をしたら<br>
Modが使えなくなるよ!<br>
このMod使うならMod使えなくなっても受け入れるし、<br>
その処罰に値する利用方法をしないよ！！<br>
ってことです。

## TownOfHost-Kってなんだい?
他のAmongUsのModとはまた一味違った、斬新で独特な機能や役職が多いModです。<br>

HostModなので部屋主のみModを導入すれば、<br>
参加者はModを導入する必要もカスタムサーバー追加等の面倒な手間なしで<br>
導入者が部屋を建て、その部屋に入ることでTownOfHost-Kを遊ぶことができます！<br>

詳しい仕様などは[Wiki](https://github.com/KYMario/TownOfHost-K/wiki)やゲーム内確認できます<br>

## 機能

### テンプレート
定型文を送信できる機能です。<br>

| タグ           | シーン                      | 対象               |
| -------------- | --------------------------- | ------------------ |
| start          | ゲーム開始のカウントダウン前 | 全員 |

#### 変数展開

| 変数名               | 内容                       |
| -------------------- | ------------------------- |
| Timer                | 部屋が落ちるまでのタイマー  |
| Roles                | 有効な役職を表示           |
| ModColor             | Modのカラーコード          |

#### クライアント設定
| オプション　　                      | 機能                                                | 使えるシーン    |
| ----------------------------------- | ------------------------------------------------- | ---------------  |
| 廃村                                | ホストが押すと廃村が行われる(非ホストだと表示されない) | ゲーム内         |
| 読み上げ                            | 読み上げのON/OFF                                    | どこでも         |
| ウェブフック                        | ゲーム開始時や終了時にWebHookが送信するかのON/OFF     | どこでも         |
| ズームを有効にする                  | 画面のサイズ？を変えれるようにするかのON/OFF           | どこでも/死亡時  |
| 読み上げの同期                      | ホストと声の設定を同期                               | どこでも         |
|推奨設定を隠す　　　　　　　　　　　　　|バニラ設定の推奨設定を消すかのON/OFF　　　　　　　    　| どこでも　　　　　|
|ボタンの見た目を変更する　　　　　　　　|ボタンの見た目をMODクライアント用にするかのON/OFF　　   | どこでも　　　　|
※廃村ボタンは左シフトと同時押しすると強制廃村となります。
どうにもできないときにお使いください。

### 読み上げ機能
棒読みちゃんを使用してチャットを読み上げることができる機能です<br>
コマンドで声を設定することができます<br>

**[使い方] /voice <音質> <音量> <速度> <音程>**<br>
**[音質一覧を確認] /voice get**


> **Note**
> - 棒読みちゃんを起動してる時のみ読み上げ設定をONにしてください (自動でOFFになります)
> - [TOHK_DATA\options.txt](#オプションファイル)からポートを設定できます。

### ウェブフック
ゲーム開始や終了時Discordにメッセージを送ってくれる機能<br>

> **Note**
> - [TOHK_DATA\options.txt](#オプションファイル)からウェブフックのURLを設定できます。

### オプションファイル
TOHK_DATAフォルダの中にあるoptions.txt<br>
[読み上げ](#読み上げ機能)や[ウェブフック](#ウェブフック)の設定ができます。

## 追加設定

### 特殊モード
メイン設定の**特殊モード**内に<br>
インポスター視点、全員の役職が分かる**インサイダーモード**や、<br>
**インポスター同士が役職を確認できる**といった設定が追加されています。<br>

**設定**<br>
|設定名|内容|
|-----|----|
|インサイダーモード|インポスター陣営に全員の役職が分かる|
|色名前モード|全員の名前がクルーの色になる|
|インポスターがお互いの役職を確認できる|その名の通り...|
|サドンデスモード|[詳しくはこちら](#サドンデスモード)|

### 試験的設定
ちょっと安定していないという設定はこちらに入れてます。<br>
試してみてバグがあればご報告お願いします。<br>

**設定**<br>
|設定名|内容|
|-----|-----|
|会議前に視界を暗転させる||
|会議後にフラッシュを入れる||
|resetcam式暗転対策|AirShipだとスポーン位置を選択できないようになります。毎ターン自身の死体が出ます。|
|試合終了にCustomRpcを使用する|パケットで落ちたらOFFにしてください|

### マップ改造
マップを調整することができます。<br>

**設定**<br>
|設定名|内容|
|-----|----|
|電気室の構造変化|会議後にAirShipの電気室をリセットします|
|昇降機のリフトを無効化|AirShipのリフトを使用不可にします|
|ベントを無効化|一定人数以下になるとベントを使用不可にします|
|会議後にドア状況をリセットする|...|
|きのこの胞子を無効化|ファングルのジャングルキノコを無効化します|
|デバイスを無効化|各情報機器を使用不可にします|
|情報機器に制限時間を設ける|[詳しくはこちら](#情報機器の利用制限)|
|┗ 会議で残り時間が分かる|各陣営毎に残り時間が見えるようになります|
|ターン毎に情報機器の制限時間を設ける|↑↑↑|
|タスクを無効化する|ONにするとタスクを排出しないようにします|

### サボタージュ設定
サボタージュ設定があります。<br>
ジャッカルが鳴らしたサボならジャッカルが勝利する**サボタージュでの勝利を変更する**設定や、<br>
停電/通信妨害から一定時間修復不可になる設定があります。

**設定**<br>
|設定名|内容|
|-----|----|
|サボタージュの時間制御|緊急サボタージュの時間制限を設定できます|
|サボタージュでの勝利を変更する|サボタージュを鳴らした陣営が勝利するようにします|
|サボタージュのクールダウン制御|サボタージュのクールダウンを設定します|
|通信妨害の特殊設定||
|┣ コミュ中のカモフラージュ|通信妨害中全員の姿が分かりません|
|┗一定時間通信妨害を修復できない|...|
|停電の特殊設定||
|一定時間停電を修復できない|...|
|展望の配電盤を無効化|...|
|昇降機の配電盤を無効化|...|
|貨物室の配電盤を無効化|...|
|配電盤妨害を無効化|既に上がっているスイッチを下げれ無くします|
|ドア閉鎖が出来ない|ドア閉めが出来なくなります|

### 情報機器の利用制限
試合全体/1ターンで情報機器に時間制限を設けることができます。<br>
見ている人数分秒数が減ります。<br>
非クライアントは周辺にとどまっているだけで時間制限が減っていくので止まるくらいなら使いましょう。<br>

### ランダムスポーン
スポーン位置をランダムに変えます。<br>
**次のスポーンポイントが通知される**設定がOnの場合、会議中〔〕内に次のスポーン位置が記載されています。<br>

### カスタムスポーン
ランダムスポーンを追加します。<br>
オンラインゲームの下にカスタムスポーンを設定というボタンがあります。<br>
そこで設定してください。<br>
ランダムスポーン内にカスタムスポーンがありますのでそれを有効にすると適応されます。<br>

### 初手強制会議
ゲーム開始直後に強制会議を行う設定です。<br>
初手ボタンを使用するのは...だったりコマンドが分からないという時におすすめ。<br>

設定がONの時はシャイボーイやウルトラスターの能力が一時的に使えなかったり、<br>
キルも行えなかったりします。<br>

**設定**
|設定名|
|------|
|初手に強制会議を行う|
|┗ 初手強制会議中にゲッサーや自投票系役職の能力が使える|

### 幽霊設定
死亡した船員がどこまで情報を得られるかの設定があります。<br>
死者のタスク免除以外幽霊役職に影響はありません。<br>

### 拡張設定
キルクール0でキルが出来るようになる設定や<br>
投票結果をターン中名前の下に表示する設定、<br>
他人のペットが見えなくなる設定、<br>
通常タスクを全員共通にしない設定<br>

などちょっと便利な設定が色々あります。

**設定**<br>
|設定名|内容|
|------|---|
|初期スポーン時のクールダウン修正|初期スポーンのキルクールを設定値にします|
|キルクール0でキルが出来る|キルク0でもキル可能になります|
|投票結果を表示する|前会議の投票結果を名前の下に表示します|
|┗道連れされた人も表示する|前会議で道連れされた人なども表示します|
|他人のペットが見えない|他人のペットが見えなくなります|
|通常タスクを共通にしない|通常タスクを全員共通にしなくなります|
|秘匿チャット|[詳しくはこちら](#秘匿チャット)|

### 秘匿チャット
会議中特定の陣営のみで会話することができます。<br>

インポスターの秘匿チャットにマッドメイトは参加できません。<br>
ジャッカルの秘匿チャットには、ジャッカルドール含むジャッカル陣営全員参加できます<br>
ラバーズの秘匿チャットではラバーズの相方とだけのチャットが出来ます。<br>

**設定**
|設定名|
|------|
|インポスター秘匿チャット(/ic)|
|ジャッカル秘匿チャット(/jc)|
|ラバーズ秘匿チャット(/lc)|
